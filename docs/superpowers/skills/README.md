# Vendored skills

These are plain-file copies of three skills from the [Superpowers](https://github.com/obra/superpowers)
Claude Code plugin (v6.1.0), vendored here so an agent without that plugin
installed — e.g. an online/cloud Claude session — can still follow
`docs/superpowers/plans/2026-07-04-staged-build-process.md` (or any future
plan in this repo) end to end.

- `executing-plans/SKILL.md` — how to work through a plan's tasks one at a time.
- `using-git-worktrees/SKILL.md` — how to get an isolated workspace before starting.
- `finishing-a-development-branch/SKILL.md` — how to wrap up once all tasks pass.

If you have the actual Superpowers plugin available, use it instead (invoke
`superpowers:executing-plans` etc.) — it may have been updated since this copy
was made. These files are a fallback, not the source of truth.
